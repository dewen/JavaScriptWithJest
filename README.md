# Jest Practice Samples

## Install

1. clone this repo
    ```
    git clone https://github.com/dewen/JavaScriptWithJest.git js_prac
    cd js_prac
    ```

1. install node dependencies
    ```
    npm install
    ```


## Run tests with JavaScript coding.


1. start watching tests

    ```
    npm run test:watch
    ```

1. start your JS coding under
    ```
    ./src/
    ```
